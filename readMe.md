#Woom

Zoom Clone using NodeJs, WebRTC and WebSockets

## 프로젝트소개
>
WebSockets와 WebRTC, NodeJs를 이용한 Zoom clone coding

## 개발기간
>
2022.11.25~2022.11.26

## 멤버
>
개인프로젝트

## 사용기술
>
- PUG
- javascript
- node.js
- express

# 주요기능
>
### 서버를 이용한 통신
- node.js를 이용해 구현
- 서버가 두 클라이언트를 연결해주는 역할.

### 통신
- WebSockets, WebRTC를 이용한 통신

# 개선사항
- 다른 프로젝트에 채팅 기능을 넣기위해 사용예정.
